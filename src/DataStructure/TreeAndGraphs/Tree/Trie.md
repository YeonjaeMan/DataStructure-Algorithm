## Trie
- 이진 검색 트리에서 어떤 노드를 검색하는 데 O(log n)의 시간이 걸린다.
- 만약, 노드들에 문자열이 들어있고 이러한 문자열을 검색한다고 한다면,
- 길이가 m인 문자열을 검색할 때 걸리는 시간복잡도는 O(m log n)이다.
- O(m log n)을 O(m)으로 만들어줄 수 있는 자료구조로 Trie(트라이)가 있다.
- Trie는 노드에 한 문자를 저장한다.

>         [Trie]                 [Tree]
>            c                     car
>           / \                    / \
>          a   o                 cat  cow        
>         / \ / \                /
>        r  t w  p             copy
>                 \
>                  y
